# 🎵 SoundDNA
SoundDNA analyzes spotify listening habits and generate AI-powered personality insights and suggestions.

## Features
- Analyze your top artists and genres
- Music personality report
- AI-generated listening insights
- Spotify integration

## Tech Stack
  - Python
  - Spotify Web API
  - Large Language Models (LLMs)
  - Streamlit
  - Git & GitHub
 
  ## Roadmap
  
  ### Phase 1 - Core Integration
  * Create Spotify Developer APp
  * Implement Spotify OAuth Login
  * Fetch User´s top artists, tracks & genres
  
  ### Phase 2 - Music Analytics
  * Build genre analysis engine
  * Calculate listening statistics
  * Generate listener archetypes
  * Create music taste scoring system

  ### Phase 3 - AI Insights
  * Integrate LLM Providers
  * Generate AI-powered personality report
  * Generate personalized music recommendations
  * Compare results across AI models
  
  ### Phase 4 - Dashboard
  * Build Streamlit dashboard
  * Visualize listening trends
  * Display genre distribution charts
  * Export reports as PDF

  ### Phase 5 - Social Features
  * Music compatibility between users
  * Shared artist analysis
  * Shared genre analysis
  * Compatibility score

  ### Phase 6 — Future Vision
  * Apple Music, Tidal and YT Music support
  * Mobile application
  * Public API for developers
